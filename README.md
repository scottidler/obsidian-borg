# obsidian-borg
YouTube and WebURL Assimilator

## Vault Setup

### Obsidian Settings

- **Editor → Readable line length**: Turn **off** so Dataview tables (Borg Dashboard) use the full window width.

### CSS Snippet

A CSS snippet is installed at `.obsidian/snippets/dataview-wide-tables.css` to make Dataview tables stretch to full width and give the Title column 60% of the space. Enable it in **Settings → Appearance → CSS snippets**.

### Required Plugins

- [Dataview](https://github.com/blacksmithgu/obsidian-dataview) — powers the Borg Dashboard queries (Today, Yesterday, This Week, This Month)
